Interactive current Browser Usage data for investigating the browsers we should be supporting. You are able to filter the list of browsers by version and see the percentage of internet users who this targets.

Use the predefined filters in the drop down or expand each browser and select specific versions of the browser.

Old versions which have a less than 1% usage are collapsed into a single option in the list of versions.

 

Browser usage data is taken from caniuse.com
